# Safety And Compliance

This skill is intended for public distribution. Keep every output shareable.

## Never Include

- API keys, tokens, passwords, cookies, or credentials.
- Internal app URLs or admin routes.
- Database IDs, bucket names, storage paths, or migration details.
- Private local paths.
- Real client files unless the user explicitly provides them for their own public use.
- Invoices, contracts, payment links, screenshots from private tools, logs, state files, or generated private media.
- Full VSL scripts or proprietary script packs.

## Public Source Rules

- Cite public URLs.
- Date your access when possible.
- Mark weak inferences as hypotheses.
- Do not claim Meta Ads duration equals proven profitability; call it a proxy.

## Publishing Gate

Before publishing a repo or generated folder:

1. Run the public safety audit script.
2. Search manually for common secret names and private URLs.
3. Confirm there are no binary client deliverables.
4. Confirm the CTA links only to `https://www.leadfactory.my`.
